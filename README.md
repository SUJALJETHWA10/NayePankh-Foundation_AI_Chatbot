# 🌱 NayePankh Foundation – AI FAQ Chatbot

An AI-powered chatbot built for the **NayePankh Foundation** to assist users with information about volunteering, donations, internships, and the NGO’s mission.

It uses **Flask + Groq LLM + Voice AI + Speech Recognition + Voice Embeddings** to create a smart conversational assistant.

---

## 🚀 Features

- 🤖 AI FAQ Chatbot (Groq LLM - LLaMA 3)
- 🔊 Voice Output (Text-to-Speech)
- 🌐 Clean Web UI (HTML + CSS + JS)
- ⚡ Fast Flask backend

---

## 🏢 About NayePankh Foundation

**NayePankh Foundation** is a non-profit organization working for:

- 📚 Education for underprivileged children  
- 🍲 Food distribution programs  
- 👕 Clothes donation drives  
- 👩‍🦰 Women empowerment initiatives  
- 🤝 Volunteer and internship opportunities  

> “Naye Pankh” means *New Wings*, symbolizing empowerment and growth.

---

## 🛠️ Tech Stack

- Python 3.14
- Flask
- Groq API (LLaMA 3)
- Pyttsx3 (Text-to-Speech)
- HTML, CSS, JavaScript
