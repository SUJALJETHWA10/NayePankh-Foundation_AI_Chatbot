# 🌱 NayePankh Foundation – AI FAQ Chatbot

An **AI-powered FAQ Chatbot** built for **NayePankh Foundation** to help users learn about the NGO, volunteering, internships, donations, and community programs.

The chatbot uses **Groq LLM (LLaMA 3)** for intelligent responses and is built with **Flask, HTML, CSS, and JavaScript**.

---

## 🚀 Features

✅ AI-powered FAQ Chatbot  
✅ Fast responses using **Groq API (LLaMA 3.3 70B)**  
✅ Information about:
- NGO Mission & Vision
- Volunteer Programs
- Internships
- Donations
- Community Welfare Initiatives

✅ Modern Web Interface  
✅ Flask Backend API  
✅ Easy to run locally

## ⚙️ Installation Steps

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/nayepankh-ai-chatbot.git

cd nayepankh-ai-chatbot
```

---

### 2️⃣ Create Virtual Environment

Windows:

```bash
py -3.14 -m venv venv
```

Activate virtual environment:

```bash
.\venv\Scripts\activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## 🔐 Create `.env` File

Create a `.env` file in the project root folder:

```env
GROQ_API_KEY=your_groq_api_key_here
```

Example:

```env
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxx
```
